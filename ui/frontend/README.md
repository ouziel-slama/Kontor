## Env
two new env vars must be added to the root .envrc
`VITE_ELECTRS_URL`
`VITE_KONTOR_URL`

## To run the app 
1. run the kontor server
`cd kontor`
`cargo run`

2. Run the fronted

# for local development
`cd ui/frontend`
`npm install`
`npm run dev`

# for running the rust frontend
`cd ui`
`cargo run`

App runs on localhost:3000
